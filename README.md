# Dictionary-App

A responsive web-based dictionary app built with HTML, CSS, and JavaScript that fetches word meanings, phonetics, and pronunciations using a public API, with a light/dark theme toggle feature.

## Features
- Search word meanings
- Phonetics & pronunciation audio
- Light/Dark theme toggle
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- Public Dictionary API
